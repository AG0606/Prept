## Overview

**Prept** is a rigorous technical interview simulation platform engineered for success. It offers real-time feedback, live code execution, and deep performance analytics built for top engineers preparing for FAANG and other top-tier technical interviews.

## Features

- **Live Code Execution**: Write, compile, and run code directly in the browser with our embedded Monaco editor.
- **Adaptive AI Voice**: Engage in real-time conversational interviews where the AI listens, interrupts, and asks deep architectural follow-up questions.
- **Deep Analytics**: Receive dimensional scores on your confidence, communication pace, and technical depth. Every pause and "um" is tracked.
- **Resume Intelligence**: Upload your PDF resume and the system ingests your background to tailor architecture questions specific to your experience.
- **Behavioral Mastery**: Train on leadership principles using the STAR framework.
- **System Design Mode**: Verbalize highly-scalable architectures while the AI comprehends load balancing, sharding, and caching strategies.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: Vanilla CSS & Tailwind CSS (Brutalist Design System)
- **Editor**: Monaco Editor
- **Speech**: Web Speech API & Web Audio API
- **AI Integration**: Gemini / Custom AI Logic
- **Database**: Prisma

## Getting Started

First, run the development server:

```bash

# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design Philosophy

The application features a custom **Brutalist Blueprint Design System**. It's built for focus, eliminating generic gradients and shadows in favor of sharp 1px borders, high-contrast monochrome layouts with striking indigo accents, and animated "system" elements like falling blocks and audio visualizers.
